# Physics-Informed Numerical Methods

A collection of computational physics and machine learning tools for solving partial differential equations (PDEs), combining classical numerical methods with modern physics-informed deep learning approaches.

## Projects

### 1. FEM Solver for Maxwell's Equations
- **Method:** Finite Element Method (FEM) using scikit-fem
- **Application:** 2D magnetostatics
- **Key Features:** Poisson equation solver, magnetic field computation

### 2. Physics-Informed Neural Network for 2D Stokes Flow
- **Method:** PINN (Physics-Informed Neural Networks)
- **Application:** Incompressible fluid dynamics
- **Key Features:** Physics loss functions, domain decomposition

### 3. Tensor Network Training with SVD
- **Method:** Tensor decomposition
- **Application:** Efficient representation of high-dimensional data
- **Key Features:** SVD-based training, compression


## Applications
- Computational electromagnetics
- Biomedical flow simulation (blood flow modeling)
- Quantum many-body systems
